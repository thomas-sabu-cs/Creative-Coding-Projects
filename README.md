# Creative Coding Projects

A curated collection of interactive visual experiments, generative art sketches, and small browser games built with **HTML5 Canvas**, **p5.js**, and **vanilla JavaScript**.

This repo is organized as a “gallery”: each folder contains a self-contained project (usually an `index.html` you can open in the browser).

> - Live site:(https://thomas-sabu-cs.github.io/Creative-Coding-Projects/)

## Highlights

- Interactive sketches and mini-games with real-time rendering loops
- Canvas + p5.js experiments exploring motion, input, randomness, and visual patterns
- Clean, beginner-friendly structure: open in browser and run
